# Contributing to prisma-ast

Thank you for your interest in contributing! We welcome all kinds of contributions, including bug reports, feature requests, documentation improvements, and code.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Install dependencies** with `npm install`.
3. **Run tests** with `npm test` to ensure everything works.
4. **Open a pull request** with a clear description of your changes.

## Guidelines

- Follow the existing code style and conventions.
- Write clear, concise commit messages.
- Add or update tests as appropriate.
- Document any new features or changes.

## Reporting Issues

If you find a bug or have a feature request, please open an issue with as much detail as possible.

## Code of Conduct

This project follows a [Code of Conduct](CODE_OF_CONDUCT.md) to ensure a welcoming environment for everyone.
